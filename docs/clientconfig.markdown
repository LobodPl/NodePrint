# Objaśnienie konfiguracji klienckiej

```
{
    "nodeId":"", // Wygenerowana wartość identyfikacyjna (używana do identyfikacji w bazie danych).
    "apiKey":"", // Funkcja skrótu bcrypt od wygenerowanej wartości klucza.
    "comPort":"", // Andres portu na którym aplikacja komunikuje się z urządzeniem końcowym
    "nodeAddress":"ws://localhost:3000"  //Adres serwera głównego
}
```
